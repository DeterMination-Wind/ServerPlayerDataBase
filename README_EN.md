# ServerPlayerDataBase

A Mindustry client mod for local player database collection and query.

## Features

- Collects player PID / UID / IP / name / server history
- Optional chat recording and UID-based chat query
- Integrity metadata verification support (`sha256-spdb-v1`)
- UID tools: merge by UID, fill missing UID manually, merge two UIDs manually
- Lightweight OverlayUI query window + standalone settings query window

## Download

Download `.jar` artifacts from Releases:

- `ServerPlayerDataBase-<version>.jar`
- `ServerPlayerDataBase-<version>-android.jar`
